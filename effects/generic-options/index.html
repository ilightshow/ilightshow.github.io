Set the brightness range (lowest/highest) the lights can use. Increasing the range width will create a show with more contrast, which is perceived as more intense.

Automatic color palettes

When enabled, the app will automatically generate color palettes for you. It will create palettes that contain either 2 complementary colors or 3 colors that are evenly spaced on the color wheel (triads).

The slider in the automatic color palettes row sets the saturation of the colors. The higher the saturation, the more intense/vibrant the colors will be, and the lower the saturation, the more pastel/dull the colors will be.

Artwork color palettes

When enabled, the app will generate color palettes based on the artwork that is playing. The app will analyze the artwork and create a color palette that matches the artwork‚Äôs colors.

Custom color palettes

When enabled, the app will use the custom color palettes you have created. You can create custom color palettes in the color palette editor.

Intensity ‚ö°Ô∏è

You can choose between 3 intensity modes:

  • Intense
  • Normal (default)
  • Chill

The intensity mode sets the intensity of the show. It affects how transitions are made, how fast the lights change. It also affects how the different effects are triggered. This new intensity selection replaces the old intensity slider.

Effects Size üìè

This option determines the average number of lights controlled at the same time.

In the FX+ effects mode, you have 3 options:

  • 1 Light: Only one light will be controlled at a time.
  • 25%: 25% of the lights will be controlled at the same time.
  • 50%: 50% of the lights will be controlled at the same time.

In the standard effects mode, you have 2 options:

  • Auto: The app will automatically choose the effects size, it either tries to split the lights in a room into 2-3 groups or it will control all lights at the same time.
  • 100%: All lights will be controlled at the same time.
