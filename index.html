How does it work?

Unlike other apps, iLightShow is not only listening to your favorite device's microphone.

iLightShow leverages an advanced algorithm to create a show, it knows in advance how to sync lights with music's beats to create a perfectly synced light show!

How do I start the synchronization?

Lights

iLightShow is compatible with Philips Hue, LIFX and Nanoleaf (Aurora, Canvas and Shapes). Start by linking the app with the lights, then select at least one of them!

Spotify

Open iLightShow, then sign into the app using your Spotify account. iLightShow will then detect the playback and start the light show!

Apple Music (only available on iOS)

Open iLightShow on iOS, then select Apple Music. You'll be asked to grant the access to the Media Library, then iLightShow will detect the playback!

What's new with iLightShow 2?

Smart algorithm

iLightShow 2 now has a new algorithm called "party mode". This algorithm allows iLightShow to detect when to trigger some specific kind of effects.

New options/effects

iLightShow 2 now has new strobe effects, new flashes and more options to create a better show.

You now have the possibility to control the brightness of each individual light.

Supports the Hue Entertainment

The Hue entertainment feature (or protocol) is awesome. It allows the app to control up to 10 lights per Hue bridge at the same time with no latency.

This feature can be hard to understand for new Hue users, which is why iLightShow automatically manages Hue Entertainment groups under the hood. Just select the lights you want to use, and iLightShow 2 will create it for you.
